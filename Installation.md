# Setup

### Materials required for setup:

- Docker image with Hyperledger Besu for local testing;
- Docker images for ZK-Rollup nodes;
- ZK-Rollup demo source code and test scripts;

All files required for the environment deployment steps can be downloaded from [this link](https://stahamsaprivacy.blob.core.windows.net/hamsa-privacy-images/Hamsa-Microsoft-Privacy.zip).

Extract the file to the desired directory and access it using a command-line terminal. Then, follow the instructions.
